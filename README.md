# sea_battle

this is one of the tasks at school 21 front-end bootcamp,
the goal is to implement sea-battle game using js / canvas / express.js

stack
- html / css / canvas
- js
- express.js
- sqlite / sequelize (ORM)

prerequisites:
1. nodejs

launch
1. npm i
2. cd src && npm start
3. open http://localhost:3000

deploy
1. serverless static version
